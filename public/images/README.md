# 画像フォルダ

AR空間に表示する透過PNG画像をここに配置します。

## 使用方法

1. 透過PNG画像を用意（推奨: 512x512px または 1024x1024px）
2. `overlay.png` として保存
3. このフォルダに配置: `public/images/overlay.png`

## 推奨設定

- **形式**: PNG（透過部分を含む）
- **サイズ**: 512x512px ～ 1024x1024px
- **ファイルサイズ**: 1MB以下推奨（モバイル最適化）
- **背景**: 透明（アルファチャンネル使用）

## サンプル画像

最初は仮の画像を使用しています。実際の画像に差し替えてください。

## 画像差し替え手順

1. 新しいPNG画像を準備
2. `public/images/overlay.png` を上書き
3. ブラウザをリロード（キャッシュクリア推奨: Ctrl+Shift+R）

## サイズ調整

index.htmlの以下の部分で調整可能:

```html
<a-image
  src="/images/overlay.png"
  width="0.5"   ← 幅を変更
  height="0.5"  ← 高さを変更
```
